Heading Hierarchy

Headings establish visual hierarchy and structure. Here's how each heading level appears:

Heading Level 1 - Main Page Title

The primary heading, typically used once per page for the main title.

Heading Level 2 - Major Section

Used for major section divisions within your content.

Heading Level 3 - Subsection

Perfect for subsections under H2 headings.

Heading Level 4 - Minor Heading

Used for further subdivisions of content.

Heading Level 5 - Small Heading

For detailed breakdowns and minor sections.

Heading Level 6 - Smallest Heading

The smallest heading, used sparingly for fine-grained divisions.

// JavaScript function example

    console.log(`Hello, ${name}!`);
    return {
        message: `Welcome, ${name}`,
        timestamp: new Date()
    };
}

// Usage
const result = greetUser('World');
console.log(result.message);
