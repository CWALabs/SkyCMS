## ðŸ“– How to Use These Documents

### For Azure Marketplace Submission

1. Review the **[AZURE_MARKETPLACE_CHECKLIST.md](./AZURE_MARKETPLACE_CHECKLIST.md)** for all required tasks
2. Ensure all legal documents are finalized and reviewed
3. Link to these documents in your Partner Center offer listing:
   - **Privacy Policy URL**: Link to PRIVACY.md
   - **Terms of Use URL**: Link to TERMS_OF_SERVICE.md
   - **Support URL**: Link to SUPPORT.md

### For Customers

After deployment from Azure Marketplace, customers should:
1. Start with **[AZURE_MARKETPLACE_QUICKSTART.md](./AZURE_MARKETPLACE_QUICKSTART.md)**
2. Review **[SUPPORT.md](./SUPPORT.md)** for help options
3. Read **[PRIVACY.md](./PRIVACY.md)** to understand data handling
4. Reference **[SECURITY.md](./SECURITY.md)** for security practices

### For Contributors

- **Report security issues**: Follow process in [SECURITY.md](./SECURITY.md)
- **Get support**: Use channels in [SUPPORT.md](./SUPPORT.md)
- **Understand legal terms**: Review [TERMS_OF_SERVICE.md](./TERMS_OF_SERVICE.md)
