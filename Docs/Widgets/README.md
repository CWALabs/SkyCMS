# Widgets

Developer docs for SkyCMS widgets. Each widget provides focused, reusable UI behavior used by the Editor and runtime views.

- [Image Widget](./Image-Widget.md) — upload/replace images with FilePond, manage alt text/title, browse library.
- [Crypto Widget](./Crypto-Widget.md) — simple AES encrypt/decrypt helpers backed by CryptoJS.
- [Crumbs Widget](./Crumbs-Widget.md) — build breadcrumb navigation from meta path and title.
- [Forms Helpers](./Forms-Widget.md) — post forms with antiforgery token and utilities.
- [Nav Builder](./Nav-Builder-Widget.md) — render a nav list using the ToC API.
- [Search Widget](./Search-Widget.md) — basic search form wiring and result rendering.
- [ToC Widget](./ToC-Widget.md) — build table-of-contents lists and breadcrumbs.

If you add new widgets, list them here and link their docs.
