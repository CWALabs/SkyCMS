# Quick Start Guide - File Manager

This quick start guide will help you get up and running with the SkyCMS File Manager in just a few minutes.

## Accessing the File Manager

1. Log in to SkyCMS Editor
2. Click the **Files** button in the toolbar
3. The File Manager opens showing your `/pub` directory

## Basic Operations

### Upload a File

**Method 1: Drag and Drop**
1. Open your computer's file explorer
2. Drag one or more files into the upload zone
3. Watch as they upload automatically

**Method 2: Browse and Select**
1. Click anywhere in the upload zone
2. Select files from the dialog
3. Click **Open**

### Create a New Folder

1. In the breadcrumb area, find the "New folder name" text box
2. Enter your folder name (no spaces, use hyphens)
3. Click **Create**

### Edit a File

**For Code Files (HTML, CSS, JS):**
1. Find your file in the list
2. Click the Monaco editor icon (looks like code brackets)
3. Edit your code
4. Press `Ctrl+S` to save

**For Images:**
1. Find your image in the list
2. Click the Filerobot icon
3. Make your edits (crop, adjust, annotate)
4. Click **Save**

### Copy a File URL

1. Find your file in the list
2. Click the **clipboard icon** next to the file
3. The URL is copied - paste it anywhere you need it

### Delete Files

1. Check the box next to each file you want to delete
2. Click the **Delete** button
3. Confirm the deletion

### Move or Copy Files

1. Select files by checking their boxes
2. Click **Copy/Move**
3. Navigate to the destination folder
4. Click **Move to here** (to move) or **Copy to here** (to copy)

## Tips for Success

- **Keep organized** - Use folders to group related files
- **Use descriptive names** - Make files easy to find later
- **Optimize images** - Resize and compress before uploading
- **Save often** - Press Ctrl+S frequently when editing
- **Check your selection** - Use "Show Selected" to review before bulk operations

## Common Tasks

### Uploading Multiple Images

1. Navigate to your images folder (or create one)
2. Select all images from your computer
3. Drag them all into the upload zone at once
4. Wait for the progress bar to complete

### Organizing Files

1. Create folders for different types of content (images, scripts, styles)
2. Use the Copy/Move function to reorganize existing files
3. Delete old or unused files regularly

### Editing Page Styles

1. Navigate to your CSS file location
2. Click the Monaco editor icon next to your `.css` file
3. Make your changes with syntax highlighting
4. Save with Ctrl+S
5. Refresh your website to see changes

### Quick Image Edits

1. Switch to thumbnail view by clicking **Show image thumbnails**
2. Click the Filerobot icon on any image card
3. Make quick adjustments or crops
4. Save and close

## Keyboard Shortcuts

- **Ctrl+S** - Save in code editor
- **Ctrl+F** - Find in code editor
- **Ctrl+Z** - Undo in code editor
