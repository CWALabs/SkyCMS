## Quick Reference

### Supported File Types

**Code Files:**
- `.html`, `.htm` - HTML documents
- `.css` - Stylesheets
- `.js` - JavaScript
- `.json` - JSON data
- `.xml` - XML documents
- `.txt` - Plain text

**Image Files:**
- `.jpg`, `.jpeg` - JPEG images
- `.png` - PNG images
- `.gif` - GIF images
- `.webp` - WebP images
- `.svg` - SVG graphics
- `.apng`, `.avif` - Modern formats
