SkyCMS

SkyCMS is a lightweight, cloud-native CMS for creating, managing, and delivering content across static, headless, and decoupled architectures—optimized for Azure with multi-cloud storage options.

Key Features

Performance

  • Static delivery via Azure Storage + CDN for instant loads
  • Efficient caching and async pipelines

Flexible Architectures

  • Static: maximum speed using blob hosting
  • Headless: API-driven content for web/mobile/apps
  • Decoupled: dynamic pages with CMS content

Multi-Cloud Ready

  • Storage: Azure Storage, Amazon S3 or Cloudflare S2
  • Databases: Cosmos DB, Azure SQL, MySQL, SQLite
  • Security: Microsoft, Google, local account authentication

Rich Editing

  • CKEditor 5 WYSIWYG, GrapesJS drag-and-drop
  • Monaco (VS Code) editor with syntax highlighting
  • File management, image editing, modern uploads

Website Features

  • SEO support, sitemap generation
  • Media library, versioned articles, navigation

Enterprise-Grade

  • Role-based access with ASP.NET Core Identity
  • Multi-tenant support, Azure Monitor integration

Developer-Friendly

  • Docker containers, "Deploy to Azure" ARM template
  • Clean .NET 9+ codebase with extensible APIs

Architecture Options

  • Static: Rendered to Azure Blob Storage
  • Headless: API delivery to any channel
  • Decoupled: Dynamic routes with CMS content

Security

  • ASP.NET Core Identity with Azure AD B2C
  • Key Vault secrets, managed identity support
  • Role-based permissions and access controls

Getting Started

  • Use "Deploy to Azure" for quick provisioning
  • Select storage (Blob) and database (Cosmos/SQL)
  • Sign in with Entra ID and start authoring

What's Included

  • Editor app (content creation and management)
  • Publisher app (high-performance delivery)
  • Shared libraries and multi-cloud abstractions

SkyCMS delivers a fast, modern CMS that's simple to deploy, easy to edit, and built for Azure—with freedom for multi-cloud growth.
