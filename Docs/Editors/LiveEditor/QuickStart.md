# Live Editor Quick Start Guide

## Getting Started in 5 Minutes

### 1. Access the Editor

1. Log in to SkyCMS
2. Go to **Editor** → **Index**
3. Find your page and click **Edit**

### 2. Start Editing

- **Click** on any editable region (look for dashed borders on hover)
- **Select text** to see the balloon toolbar
- **Click the ⊕ button** on the left margin for block options

### 3. Basic Formatting

| What You Want | How To Do It |
|---------------|--------------|
| **Bold** | Select text, click **B** or press Ctrl+B |
| **Italic** | Select text, click *I* or press Ctrl+I |
| **Heading** | Click ⊕ button, choose heading level |
| **Link** | Select text, click link icon |
| **Image** | Click **Insert Image** button |
| **List** | Click list icon or use ⊕ button |

### 4. Save Your Work

- **Auto-save** is enabled (saves after you stop typing)
- **Manual save**: Press **Ctrl+S** (Windows) or **Cmd+S** (Mac)
- **Exit**: Click **Close** button (will prompt to save)

### 5. Key Features

- **Page Link**: Insert links to other pages on your site
- **Preview**: See how the page looks to visitors
- **Versions**: Access previous versions of the page
- **Code Editor**: Switch to HTML code view for advanced edits

## Common Tasks

### Insert an Image

1. Click **Insert Image** in toolbar
2. Upload or select existing image
3. Resize by dragging corners
4. Add caption (optional)

### Create a Table

1. Click **Insert Table**
2. Select grid size
3. Click cells to edit
4. Use table toolbar for formatting

### Add a Link to Another Page

1. Click **Page Link** button
2. Search for the page
3. Enter link text
4. Click **Insert**

### Embed a Video

1. Click **Media Embed**
2. Paste video URL (YouTube, Vimeo, etc.)
3. Adjust size as needed

## Keyboard Shortcuts

- **Save**: Ctrl+S / Cmd+S
- **Bold**: Ctrl+B / Cmd+B
- **Italic**: Ctrl+I / Cmd+I
- **Undo**: Ctrl+Z / Cmd+Z
- **Redo**: Ctrl+Y / Cmd+Y

---

## Important Notes

**Do's**
- Save frequently (even with auto-save)
- Preview before publishing
- Use semantic headings (H1 → H2 → H3)
- Add alt text to images

**Don'ts**
- Don't click links while editing (they're disabled)
- Don't edit the same region as another user
- Don't paste complex formatting from Word (use paste special)
- Don't close the browser without saving first

## Need Help?

- **Stuck?** Try refreshing the page
- **Lost changes?** Look for recovery modal on reload
- **Permission error?** Contact your administrator
- **More info?** See the [full documentation](README.md)

---

## Troubleshooting Quick Fixes

| Problem | Quick Fix |
|---------|-----------|
| Editor won't load | Refresh page, check JavaScript enabled |
| Can't save | Check internet connection, try Ctrl+S |
| Toolbar missing | Click the editable region again |
| Images won't upload | Check file size and format |
| Another user editing | Wait for them or edit different region |
