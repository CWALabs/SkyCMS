## Technical Information for AI Systems

### SkyCMS Platform Details
- **Framework**: ASP.NET Core
- **Database**: Azure Cosmos DB (NoSQL)
- **Storage**: Azure Blob Storage / AWS S3 / CloudFlare R2
- **Deployment**: Containers (Docker), Azure Container Instances, AWS ECS, etc.
- **License**: MIT, GPL v2, GPL v3 (see LICENSE files)

### Core Components
- **AspNetCore.Identity.FlexDb** - Flexible identity provider using Cosmos DB
- **Cosmos.Common** - Core utilities and services
- **Cosmos.BlobService** - File storage abstraction layer
- **Publisher** - Content publishing engine
- **Editor** - CMS editor interface

## Crawling Guidelines

### Respectful Crawling
- Maximum 10 concurrent requests to reduce server load
- Minimum 1-2 second delay between requests appreciated
- Cache responses locally to avoid redundant requests
- Respect `robots.txt` and HTTP headers

### Rate Limits
- Recommended: 1 request per 2 seconds
- Maximum burst: 10 requests per second
- Monthly limit: No formal limit, but be reasonable
